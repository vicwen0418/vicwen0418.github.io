手把手超詳細教學：想擁有GitHub酷炫的個人頁面嗎？

Introduction - 前言

今天要來介紹GitHub的個人頁面（Profile），相信大家點到某些大神的帳號時會看到有豐富的GitHub插件以及令人熱血的自我介紹，你可能會好奇那是如何辦到的？沒關係，今天就來帶大家了解，讓剛踏入GitHub的小白們也能有美美、帥氣爆棚的個人版面（雖然我也是小白），並且帶給你一些其他大神做的神奇插件。

今天便以我的做範例：

Establish - 建立

首先，在布置我們的Profile之前，我們需要先建立屬於我們自己的頁面，不然是要布置什麼？

  • 登入GitHub
  • 點右上角的 「New repository」 (建立倉庫)

  • 名稱請輸入「自己的帳號名稱」
    因為我建立過了，所以會出現紅色的警告字串，但你不會，所以請放心的點擊「Create repository」， 以及記得要設定公開。

進到Repository頁面後直接點下方的「Initialize this repository with a README」，便會新增一個README.md的檔案，你只需要寫markdown語法或是HTML在檔案裡面，寫完後按下「Commit changes」儲存，這樣就會顯示在你的Profile上了。

  • 若只是要求基本功能，簡單的打些字，放圖片，可以使用markdown語法。

  • 若追求排版，想排的與眾不同，可以使用HTML來寫。

寫markdown我推薦用HackMD平台來寫，很方便，圖片複製直接貼上去，會幫你丟到imgur自動上傳，轉成網址給你，不用再自己轉換。

若你真的超懶，可以直接用Profile Generator，幫你產生你的README。

real substences - 乾貨

置中

加上 align="center"，即可達到置中的效果。

Badge（徽章）

徽章是利用Shields.io網頁提供的參數製作。

Badge可以用在介紹軟體版本、程式語言、相關專案...等等，

我們可以將網址參數拆分成以下幾項，參數中間會用'&'分隔開。
* message：文字端（右方）的文字！

  • logo：就logo XD，可參照Simple Icons，或是此Repository

  • labelcolor：label的背景顏色，可參考色碼表，記得要把'#'去掉。

  • color：message（右方）的背景顏色

  • logocolor：label（左方）的logo顏色

  • label：label（左方）的文字

  • style：參照下圖。

GitHub stats

若你想要展示你的Repository在主頁、展示你的commit、PR數量、使用的語言，想跟別人炫耀！那就可以利用這個Repository做製作！作者提供了許多功能插件，甚至還能選擇你喜歡的主題，超U質。

GitHub Stats Card

參數：
* username：你的帳號名稱

  • repo：你的repository名稱

  • theme：主題（點選作者的Repository做選擇）

GitHub Extra Pins

參數：
* username：你的帳號名稱

  • repo：你的repository名稱

  • theme：主題（點選作者的Repository做選擇）

建議可以加個超連結使訪客點擊自動跳轉，否則他只是張圖片。

Top Languages Card

  • Compact Language Card Layout

  • Hide Progress Bars

  • Show more languages

  • Hide individual languages / repositories

你可以加上 &exclude_repo=repo1,repo2來隱藏repositories，或是加上&hide=language1,language2來隱藏語言。
